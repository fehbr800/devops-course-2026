## Construção de String de Busca

Para a realização da busca de artigos científicos, é fundamental construir uma **string de busca bem estruturada**, utilizando palavras-chave, operadores booleanos (AND, OR) e sinônimos.

 Utilize o guia abaixo para aprender como montar sua string de busca:

🔗 https://library.cranfield.ac.uk/literature-review/search-strings

---

###  Dicas importantes

- Use operadores booleanos:
  - **AND** → combina termos (reduz resultados)
  - **OR** → inclui sinônimos (aumenta resultados)

- Utilize parênteses para organizar a lógica:

- (DevOps OR "Continuous Integration") AND ("Automated Testing" OR "Software Testing")


- Sempre escreva os termos em **inglês**, pois as principais bases científicas utilizam esse idioma.

- Teste e refine sua string de busca até obter resultados relevantes.

---

### Objetivo

Criar uma string de busca eficiente permite:

- Encontrar artigos mais relevantes
- Reduzir ruído na pesquisa
- Melhorar a qualidade da revisão de literatura
